# Character pack (CC0)

Low-poly rigged characters with baked Idle / Walk / Run clips. Self-contained .gltf
(mesh + skeleton + animations embedded). Used for the Explore avatar (Worker) and the
villager NPCs (systems/villagers.js), cloned per-instance with SkeletonUtils.

License: CC0 (public domain). Drop-in via src/assets/manifest.js (char_* ids).
